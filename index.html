1. Cài đặt môi trường lập trình với smart contract với Near

1.1. Cài đặt Rust

  • Linux/MacOs
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
  • Window: Hướng dẫn cài đặt môi trường lập trình Rust. Youtube VBI Academy click

1.2. Cài đặt Near CLI

  • Linux/MacOs

    • cài đặt npm và node theo distro click
        npm install -g near-cli
  • Window
    • Cài đặt WSL
    • Cài đặt npm
    • Cài đặt Node.js
    • Thay đổi npm directory mặc định.Tránh vến đề về quyền với WSL Click

    • Mở WSL và chạy lệnh 

            npm install -g near-cli

1.3. Cài đặt cargo-near

cargo install cargo-near

1.4. Tương tác với NEAR smart contract

cargo near --help

  • Tạo tài khoản Near 

        cargo near create-dev-account

  • Khởi tạo dự án mới 

        cargo near new

  • Deloy contract 

        cargo near deploy

  • Function call 

        near call --help

        near call <contractName> <methodName> [args]
